The audit ledger

Every run. On the record. Provably.

Each event is hash-chained with SHA-256. Every 100 events, the Merkle root is signed with Ed25519. Verify the whole chain offline — tampering means breaking SHA-256 or forging a signature, not editing a row.

Frequently asked

Straight answers.

What is Tracelane?

An open-source (Apache 2.0) flight recorder for AI agents: a universal Rust LLM gateway, full-fidelity OpenTelemetry observability, and a tamper-evident audit ledger. Each event is SHA-256 hash-chained and the Merkle root is Ed25519-signed every 100 events, so any trace can be verified offline.

How is it different from Helicone, LangSmith, or Langfuse?

Tracelane unifies the gateway, observability, pre-flight policy, and a tamper-evident audit ledger in one Rust binary. The others are observability-first — you bring your own gateway and guardrails. No competitor in this category offers cryptographic audit-log integrity at the trace level.

What's the latency overhead?

Gateway overhead targets p50 <5ms, p95 <15ms, p99 <25ms (excluding provider time). Pre-flight policy targets p99 <50ms on commodity CPU. These are engineering targets — measured numbers publish as the Tracelane Reliability Benchmark v1.0 at launch.
